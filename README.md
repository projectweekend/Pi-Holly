Pi-Holly
========

A home server project for my Raspberry Pi.

## API End Points

#### Get latest system temperature reading

**GET:** `/api/system-temperature-data`

**Response:** 
```
 {
     date: "2014-01-10T23:22:24.150Z",
     celsius: 40,
     fahrenheit: 104,
     _id: "52d080b03a62e79fbf000003",
     __v: 0
 }
```
